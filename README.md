# 🎮 2048 游戏 - 完整版

一款基于 Web 的经典 2048 游戏，支持多种难度、主题切换、成就系统等功能。

## 🎯 游戏特性

### 核心功能
- ✅ 三种难度模式：简单 (5×5)、普通 (4×4)、困难 (3×3)
- ✅ 键盘和触屏双端操作支持
- ✅ 分数计算与最高分本地存储
- ✅ 游戏胜负判定

### 扩展功能
- 🎨 **主题切换**：经典/暗夜主题
- 🔊 **音效系统**：Web Audio API 实现游戏音效
- 🏆 **排行榜系统**：支持玩家昵称、按分数/时间排序、难度筛选
- 🏅 **成就系统**：首次通关、满分挑战、速度之王
- 📤 **分享功能**：一键复制成绩到剪贴板
- 📄 **个人简历**：从暂停菜单可跳转查看

### 视觉体验
- 🚀 游戏启动加载动画
- 🎉 成就解锁通知
- ✨ 方块合并动画
- 📱 响应式设计，适配手机端

## 🎮 操作说明

### 键盘操作
- **↑** / **W**：向上移动
- **↓** / **S**：向下移动
- **←** / **A**：向左移动
- **→** / **D**：向右移动

### 触屏操作
- 在游戏区域滑动手指控制方向

## 🏅 成就系统

| 成就 | 图标 | 描述 |
|------|------|------|
| 首次通关 | 🏆 | 第一次达到 2048！ |
| 满分挑战 | ⭐ | 在困难模式下达到 2048！ |
| 速度之王 | ⚡ | 在 1 分钟内达到 1024！ |
